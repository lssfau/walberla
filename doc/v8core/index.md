# V8 Core Library {#v8core-index}

> [!warning]
> The V8 core library is under active development and considered *experimental*.
> APIs may change rapidly and without notice.

The novel **V8 core library** constitutes our forthgoing effort to modernize the waLBerla framework;
in time, it will replace the various legacy core modules.

**For Users:**
Library users can find information on working with the V8 core library in the
\subpage v8core-manual "user manual".
For further information, also refer to the [example apps](#example-apps) and the [API documentation](#v8core).

**For Developers:**
Developers should start by reading the \subpage v8core-contrib "contributors guide".
